Arabic Fonts

The procedure to get Internet Explorer/Firefox/Opera to render the arabic fonts properly in a clean Windows installation appears to be:

  • Go to Start - Settings - Control Panel - Regional Options
  • Tick the box 'Arabic' at 'Language settings for the system' and click OK
  • It will now need the Windows Installation disk to install the arabic fonts.
  • Reboot to enable the fonts

In Linux (Kubuntu Fiesty Fawn), there seem to be 2 options:

  • Konqueror seems to be the only one capable of displaying readable arabic text without any extra font installations. The fonts doesn't look very nice though.
  • You can manually copy the Windows 2000 Arabic fonts in "C:\Windows\Fonts" from a windows installation to the linux fonts directory such as /usr/share/fonts/truetype/win2k-arabic-fonts/. Here is a list of the files you would need to copy: 
      andlso.ttf
  artrbdo.ttf
  artro.ttf
  simpbdo.ttf
  simpfxo.ttf
  simpo.ttf
  tradbdo.ttf
  trado.ttf
    After they have been copied, both Konqueror and Firefox seem to be able to display the arabic text the same as Internet Explorer.
