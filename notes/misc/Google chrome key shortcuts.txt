space = scroll down a page (+shift is up)

ctrl + tab = next tab (+shift = previous) or ctrl + pgup/dwn
ctrl + w = close current tab (+shift = close window)
ctrl + shift + t = restore closed tab (can do multiple times)
ctrl + num = go to that tab in order (9 is last tab)

alt + left = go back in current tab (right is go forward again)

ctrl + d = bookmark
ctrl + h = history
ctrl + j = downloads

ctrl + click = open link in new tab (ctrl + shift click = open in new window)

alt + d = go to omnibox (can also use ctrl+e/l/k)

ctrl + r = refresh (+shift = without cache)

ctrl + t = new tab
ctrl + n = new window (+shift = incognito)

ctrl + (+/-) = zoom (ctrl + 0 is reset zoom)




alt + f4 = shutdown (may close windows first)
ctrl + u = view page sourcecode




tab = move along items






alt + f = menu
alt + enter = (open new tab with previously written url)
				



f11 = fullscreen




ctrl + shift + b = show/hide bookmarks bar
ctrl + shift + o = open bookmarks manager in new tab
